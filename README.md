# Proxmox-LDC

Proxmox Linux Desktop Client

An app for the linux desktop to interact with a Proxmox server With Proxmox-LDC (Linux Desktop Client) you can connect to a Proxmox sever and interact as followed

* View the stats of the server
* View the list of Virtual Machines & Containers to see whats running and what isn't
* View and check status of network adaptors
* Enter shell to make configurations on server
* Connect to Proxmox server via API & SSH

* Stop and start vm's & Containers
* Open vim's for viewing and interacting in app
